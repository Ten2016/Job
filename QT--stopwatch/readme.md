##题目要求
------

        图中大表盘黄色指针为总时间的秒针，蓝色指针为当前计次时间的秒针，小表盘黄色指针为分针。表盘中间显示的时间为总时间，表盘左边为“计次/复位”按钮，右边为“启动/停止”按钮，根据不同的状态显示不同的文字。
开始时，“计次/复位”按钮无效，点击“启动”按钮，秒表开始工作，左侧按钮显示文字变为“计次”并且可用，右侧按钮显示文字变为“停止”。
在秒表启动后，点击左侧“计次”按钮，可在下方列表增加时间计次（上限 100次），下方时间计次以先后顺序排列，最早的计次在最下方，可通过鼠标中键滚轮上下滚动显示。点击右侧“停止”按钮，可暂停秒表，并且左侧按钮显示文字变为“复位”，右侧按钮显示文字变为“启动”。
在秒表暂停后，点击左侧“复位”按钮，将清零秒表时间和下方计次列表，左侧按钮显示文字也变为“计次”并且无效。点击右侧“启动”按钮则恢复暂停的秒表继续运行，左侧按钮显示文字变为“计次”，右侧按钮显示文字变为“启动”。
特别要求：不得使用图片来绘制表盘，窗口缩放时表盘也应能随之自动缩放，表盘和指针都不应有明显锯齿。
扩展1：列表显示的计次时间（不含当前计次），最短时间以绿色显示，最长时间以红色显示。点击表盘可以在数字模式和表盘模式之间切换。
